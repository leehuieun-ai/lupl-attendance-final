# 러플 근태관리 시스템 BigTech Final

처음부터 러플 내부 HR/근태 운영 시스템으로 설계한 버전입니다.

## 핵심 기능

- 사번 로그인
- 관리자 생성 계정만 사용
- 초기 비밀번호: `lupl + 휴대폰 뒷번호 4자리`
- 직원별 등록 가능 기기 기본 3대
- GPS + IP + 등록 기기 기반 출퇴근
- 근무지 자동 인식 및 관리자 승인
- 개인정보·위치정보 동의서 + 서명
- 비활성 직원 숨김 및 비활성 메뉴
- 연차 자동 계산
- 반차 / 시간차 / 특별휴가 / 대체휴가 / 보상휴가
- 추가 근무 수당 대신 대체휴가 적립
- 추가 근무 승인 시 휴가 잔여 자동 증가
- 휴게 시작/종료
- 근태 예외함
- 보고서 Excel 다운로드
- Apple/Google 스타일의 큰 카드형 UI

## 보안

- 카카오 REST API 키는 GitHub에 없음
- Supabase service_role key는 GitHub에 없음
- 중요 키는 Supabase Secrets에만 저장
- 직원 비밀번호는 평문 저장하지 않음
- 관리자도 비밀번호 조회 대신 초기 비밀번호 발급/재설정 방식 권장
- Supabase RLS 적용
